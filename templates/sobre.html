Sobre o Projeto

O Gerador de Piadas é um projeto desenvolvido para trazer diversão e humor de forma simples e acessível. Utilizamos a JokeAPI para buscar piadas aleatórias e garantir que todas sejam adequadas para todos os públicos.

Este projeto foi criado como parte de um trabalho escolar, com o objetivo de integrar tecnologias modernas, como o Flask para o backend e o Tailwind CSS para o design da interface.

Se você tiver alguma sugestão ou quiser contribuir com o projeto, sinta-se à vontade para entrar em contato!

Tecnologias Utilizadas

  • Flask (Backend)
  • Tailwind CSS (Design)
  • JokeAPI (Integração de Piadas)
  • Google Translate API (Tradução)
